# Sales Analytics & Business Intelligence Dashboard

## Project Overview

This project analyzes sales data using Python, MySQL, Power BI, and DAX to identify business trends, customer behavior, and product performance.

The project covers the complete data analytics workflow including data cleaning, exploratory data analysis, SQL-based business analysis, and interactive dashboard development.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MySQL
- SQL
- Power BI
- DAX

---

## Project Workflow

### 1. Data Cleaning & Preprocessing
- Checked missing values
- Removed duplicate records
- Verified data types
- Performed date transformation
- Created additional analytical features

### 2. Exploratory Data Analysis (EDA)
- Sales trend analysis
- Country-wise sales analysis
- Product line analysis
- Customer analysis
- Revenue distribution analysis

### 3. SQL Business Analysis
- Total sales calculation
- Customer ranking
- Product performance analysis
- Country-wise revenue analysis
- Running totals and growth analysis

### 4. Power BI Dashboard
- KPI cards
- Sales trends
- Product performance
- Customer insights
- Geographic analysis
- Interactive filters and slicers

---

## Key Insights

- Identified top-performing customers based on revenue.
- Analyzed sales performance across countries and product categories.
- Evaluated deal size contribution to overall revenue.
- Monitored sales trends over time using interactive dashboards.
- Generated business insights for data-driven decision-making.
